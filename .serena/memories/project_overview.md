# Project Overview
Static landing page for coffee machine repair services in Moscow. Current implementation uses plain HTML, CSS, and JavaScript without a build system. The target design is a high-converting, modern light landing page matching the provided visual reference layout, with placeholder image blocks instead of real photography.

Main files:
- `index.html` page markup
- `style.css` visual styling and responsive layout
- `script.js` interactions, sliders/tabs/timer/forms
- `prompt.md` contains the Russian content/design brief
- `assets/` exists but is currently empty
