# Style And Conventions
Use plain static web conventions: semantic HTML sections, BEM-like class names where useful, CSS variables for theme colors, and vanilla JavaScript. Keep text/content in Russian for the landing page. Design direction: light premium coffee-service landing page with white base, warm coffee/orange accents, rounded cards, subtle shadows, dashed placeholder media boxes, smooth hover states, and responsive layouts.

No framework conventions are present. Prefer concise comments only for non-obvious JS logic.
